data automation

Overview

Data Automation is a tool that cleans and merges Excel files with inconsistent headers using Python, Pandas, and a FastAPI backend. The front-end UI lets users drop in files and download a perfectly structured output.

Problem

Teams were manually fixing spreadsheets every week — dragging columns around, renaming headers, and re-typing values. It was error-prone and boring.

Solution

A scriptable pipeline that auto-detects formats (Type A / Type B), aligns headers, and outputs a clean dataset without anyone touching Excel.
